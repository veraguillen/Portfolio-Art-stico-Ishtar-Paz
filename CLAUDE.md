\# CLAUDE.md — Portafolio Artístico Inmersivo



🎭 Perfil de la Artista
Directora escénica y artista visual. Su obra se sitúa en la intersección del teatro de objetos, la naturaleza y la retro-tecnología (basura tecnológica reutilizada). Explora la materialidad de lo obsoleto y lo orgánico en espacios inmersivos.

⚙️ Entorno de Desarrollo
Frontend: Next.js 15+ (App Router) + React 19.

Estilos: Tailwind CSS v4.

Animaciones: Framer Motion (para comportamiento físico/orgánico).

🎨 Directivas Estéticas: "El Escenario Ensamblado"
Evitar interfaces estériles o minimalismo genérico. La web debe sentirse como un monitor antiguo encontrado en un bosque.

1. Atmósfera y Color (Óxido y Neón) 🛢️
Fondo: Negro puro (#000000) con una capa de textura .film-grain (ruido analógico) siempre visible.

Texto: Blanco roto (#F5F5F5) para simular fósforo o papel viejo.

Acento: Cian Fósforo Retro (#08F7FE). Usar solo para interacciones de "encendido" (hovers, botones activos).

Materia: Usar el tono Tierra/Óxido (#3D2314) para separadores o bordes sutiles.

2. Tipografía y Estructura Disruptiva 📟
Títulos: Monospace (Geist Mono o similar) en mayúsculas, con kerning (espaciado) muy amplio (ej. tracking-[0.2em]). Deben parecer etiquetas de inventario o código de terminal.

Cuerpo: Sans-serif geométrica limpia para legibilidad en textos largos.

Bordes: Evitar bordes redondeados perfectos. Usar bordes dobles o que simulen marcos de monitores CRT o soldaduras toscas.

3. Animaciones: "Vida Analógica" ⚡
Comportamiento: Nada de desplazamientos suaves genéricos. Usar efectos de parpadeo (flicker), glitches sutiles al pasar el mouse y entradas secuenciales (como si los circuitos se estuvieran cargando).

Física: Los objetos/imágenes deben tener una inercia "pesada" al moverse.

🏗️ Estilo de Código
Componentes: Funcionales de React. Priorizar la composición.

Semántica: El código debe reflejar la estructura de una obra: Hero es el "Prólogo", el Grid es el "Escenario".
